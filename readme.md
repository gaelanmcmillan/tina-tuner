# tina tuner

A tiny chromatic tuner for the command line.

## usage

### select an audio source

```
~ tina
-: tina tuner v1.0 :-
please select an audio input device
- (0) airdrop cute doggy only Microphone
- (1) BlackHole 2ch
- (2) MacBook Pro Microphone

>>> 2
```
